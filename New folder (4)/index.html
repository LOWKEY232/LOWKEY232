Network Architecture Models

Understanding the foundational models of computer networking

OSI Model (Open Systems Interconnection)

The OSI model is a conceptual framework used to understand network interactions in 7 distinct layers. Each layer has a specific function and communicates with the layers above and below it.

  • Application Layer: Provides network services to user applications
  • Presentation Layer: Translates, encrypts, and compresses data
  • Session Layer: Establishes, manages, and terminates sessions
  • Transport Layer: Provides reliable data transport and flow control
  • Network Layer: Handles logical addressing and routing
  • Data Link Layer: Provides node-to-node data transfer and error detection
  • Physical Layer: Transmits raw bit stream over physical medium

TCP/IP Model (Internet Protocol Suite)

The TCP/IP model is the practical implementation used for internet communications, consisting of 4 abstraction layers.

  • Application Layer: Combines OSI application, presentation, and session layers
  • Transport Layer: Provides end-to-end communication with TCP and UDP
  • Internet Layer: Handles addressing, packaging, and routing (IP)
  • Link Layer: Combines OSI data link and physical layers

TCP/IP is the foundation of modern internet communications and is more concise than the OSI model.

Essential Networking Protocols

Key protocols that enable communication across networks

TCP

Transmission Control Protocol provides reliable, ordered, and error-checked delivery of data between applications.

UDP

User Datagram Protocol provides connectionless communication with no guarantee of delivery, used for time-sensitive applications.

IP

Internet Protocol is responsible for addressing and routing packets across network boundaries.

HTTP/HTTPS

Hypertext Transfer Protocol (Secure) for communication between web browsers and servers.

FTP

File Transfer Protocol for transferring files between client and server on a network.

SMTP

Simple Mail Transfer Protocol for sending and routing email between mail servers.

DNS

Domain Name System translates domain names to IP addresses for locating internet resources.

SSL/TLS

Secure Sockets Layer/Transport Layer Security protocols for secure communications.
